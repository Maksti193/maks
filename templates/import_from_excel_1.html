Импорт данных из Excel

Файл должен содержать заголовки столбцов в первой строке

Инструкция по импорту:

  1. Подготовьте файл Excel с данными
  2. Первая строка должна содержать заголовки столбцов
  3. Соответствующие столбцы будут автоматически сопоставлены с полями системы
  4. Поддерживаемые форматы: .xlsx, .xls
